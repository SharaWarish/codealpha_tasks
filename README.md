# codealpha_tasks

**CodeAlpha June Batch Internship – Bioinformatics Track**

This repository contains my completed task submissions for the CodeAlpha June 2026 Bioinformatics Internship Batch. Each task is documented as a standalone report covering objectives, methodology, results, and analysis.

## 📌 Tasks Completed

### ✅ Task 1: DNA/Protein Sequence Analysis
**BLAST Homology Analysis of RecA Protein from *Deinococcus radiodurans***
Retrieved the RecA protein sequence (GenBank BAA21330.1) from NCBI and performed a BLASTP homology search against the PDB database. Documented alignment scores, percent identity, E-values, and conserved domains (ATP-binding site, hexamer interface) across bacterial and archaeal homologs.
📄 File: [`TASK 1 DNA Protein Sequence Analysis.docx`](./TASK%201%20DNA%20Protein%20Sequence%20Analysis.docx)

### ✅ Task 2: Multiple Sequence Alignment (MSA)
**Conservation Analysis of Hemoglobin Subunit Beta (HBB) Across Species**
Retrieved five reviewed (Swiss-Prot) HBB sequences from UniProtKB across different mammals and birds, performed a Multiple Sequence Alignment, and identified conserved regions and motifs reflecting evolutionary conservation of the hemoglobin beta chain.
📄 File: [`Task 2 Multiple Sequence Alignment (MSA).docx`](./Task%202%20Multiple%20Sequence%20Alignment%20%28MSA%29.docx)

### ✅ Task 3: Protein Structure Prediction
**3D Structure Prediction and Visualization of Hemoglobin Subunit Beta (HBB_HUMAN)**
Selected Hemoglobin subunit beta (UniProt P68871, HBB_HUMAN) — continuing from the Task 2 alignment set — predicted its 3D structure using AlphaFold DB, and visualized/analyzed it in PyMOL, including backbone dihedral (phi/psi) angle analysis.
📄 File: [`TASK 3 Protein Structure Prediction.docx`](./TASK%203%20Protein%20Structure%20Prediction.docx)

### ✅ Task 4: Mini Project – Bioinformatics Databases and Tool Study
**Role of Bioinformatics in Drug Discovery**
A mini project studying how computational tools, biological databases, and algorithms are used in modern drug discovery — covering target identification, compound screening, and drug interaction prediction to reduce cost and time in developing new medicines.
📄 File: [`TASK 4 Mini Project Bioinformatics Databases and Tool Study.docx`](./TASK%204%20Mini%20Project%20Bioinformatics%20Databases%20Tool%20Study.docx)
